# models/templates/

Blank model frameworks — the starting points (Stage 1).

This is where the master `performance-ratios-template.xlsx` and any other unpopulated frameworks live. Treat everything here as read-only in spirit: never fill a template in place. Copy it into `models/builds/` first, then enter data into the copy. If a template's formulas get improved, the upgrade happens here, deliberately, and gets a commit that says so.
