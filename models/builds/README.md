# models/builds/

Populated, working models — templates with real data in them (Stage 3).

Each file here started life as a framework from `models/templates/` and now carries actual company figures and live formulas. This is the "in production" folder, to borrow the product term. Name files so the company and version are obvious at a glance — e.g. `acme-performance-ratios-v2.xlsx` — so iterations don't get confused with each other.
