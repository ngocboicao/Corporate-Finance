# docs/decisions/

Decision memos — the "why we went this way" record (Stage 2).

Coming from a design background, I'm used to documenting the rationale behind a choice, not just the choice itself. Same discipline here: each memo lays out the question, the options on the table, the trade-offs, and the call I made. Six months out, this is what lets me — or anyone auditing the work — reconstruct the thinking without guessing.

Naming: `YYYY-MM-DD-short-topic.md` so the folder reads chronologically.
